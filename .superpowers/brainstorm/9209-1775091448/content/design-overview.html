🔌 하드웨어 연결

MFRC522 → Sensor 보드 (SPI)
MFRC522  →  Pico 2W
SDA (CS)  →  GP5
SCK      →  GP2
MOSI     →  GP3
MISO     →  GP4
RST      →  GP0
3.3V / GND

📡 MQTT 데이터 흐름

Sensor 보드 카드 태그 감지
  → fitpico/rfid/uid 발행  {"uid":"A3:B2:C1:D0"}

Dashboard 보드 UID 수신
  → 플래시에서 uid→사용자 매핑 조회
  → 현재 사용자 전환
  → fitpico/rfid/user 발행  {"name":"박찬웅","uid":"..."}

Display 보드 사용자명 수신
  → LCD 1행에 "HI CHANWOONG" 표시
